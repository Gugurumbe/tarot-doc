Description détaillée

La classe d'événements.

Elle contient son type, de type Evenement::Type (ça fait beaucoup de type), et le message, de type Protocole::Message. Ces deux attributs sont constants. On peut déclarer un événement via le premier constructeur (Evenement::Evenement(enum Type, const Protocole::Message &)), via le constructeur de copie (Evenement::Evenement(const Evenement &)), ou grâce à l'une des deux méthodes statiques Evenement::ClientDit ou Evenement::ServeurDit.
